# Mini Zillow - Full Stack Real Estate Platform

## Project Overview

Mini Zillow is a minimal fullstack real estate web platform inspired by Zillow.  
Users can browse property listings, view detailed information with images, and admins can manage listings through a protected dashboard.

---

## Features

- User Signup and Login with JWT authentication  
- Admin dashboard with protected routes  
- Property creation, update, deletion, and listing  
- Property detail pages with image galleries  
- Image upload integrated using Cloudinary and multer  
- Property bookmarking and rating features  
- Filter properties by category and search by location  
- Responsive UI built with Next.js and TailwindCSS  
- Form validation and real-time notifications  

---

## Tech Stack

- Frontend: Next.js, React, TailwindCSS  
- Backend: Express.js (Node.js) API  
- Database: MongoDB (MongoDB Atlas)  
- Authentication: JSON Web Tokens (JWT)  
- Image Upload: Cloudinary  
- Deployment: Vercel (Frontend), Railway/Render (Backend)
